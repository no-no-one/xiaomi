git init 是在当前目录初始化一个git仓库（运行一次）
git status 查看当前git仓库的状态
git add one.txt 将one.txt文件添加到缓存区
git commit -m “注释/日志” 将缓存区文件提交到仓库
git log 查看日志
git log --oneline 显示简洁的日志
git reset --HEAD filename  将指定文件从缓存区取回
git reset --hard comitID 回退到指定的版本
git reflog 查看所有操作
git branch 查看当前分支
git checkout branchName 切换到指定的分支
git checkout -b mhe  切换并创建新分支
git merge jskaifa  合并制定分支到当前分支
git add . 添加所有
git push -u origin master  提交到远程码云，将本地仓库修改推送到远程仓库  -u 关联
git remote add origin “远程仓库地址”  在克隆和下载找到地址，将本地仓库连接到远程仓库
git pull 拉取当前远程操作的更新
git clone “远程仓库地址”将远程仓库克隆到本地，如果你有权限，你可以push代码到远程，否则不能
git branch -a 查看本地和远程所有分支
